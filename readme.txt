~~~~~~~~~~~~~~~~~~~~~
Hemingway Child Theme
AJ Dyrbye
~~~~~~~~~~~~~~~~~~~~~

A child theme of Anders Noren's Hemingway theme for WordPress. Original theme is located at:

https://wordpress.org/themes/hemingway/

This child theme was developed over March and April of 2015 by AJ Dyrbye for use on personal website http://dyrbye.ca

Includes customizations to the CSS and minor changes to functions, header and pages. Also adds a landing page (not present in original theme) and a heavily customized 404 page at this time.
